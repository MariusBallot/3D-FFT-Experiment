## Description

A quick 3D experiment to visualize music frequencies

## Requirements

```
NPM
Webpack 3.0 and up
```

## Usage

```
git clone git@github.com:MariusBallot/3D-FFT-Experiment.git
npm i
```

### Launch the experiment

```
npm start
```

Click the screen to launch
